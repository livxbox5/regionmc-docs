🏗️ Основные флаги защиты

ФлагОписаниеЗначения
buildРазрешить/запретить строительство (размещение блоков)allow / deny
breakРазрешить/запретить разрушение блоковallow / deny
interactВзаимодействие с объектами (кнопки, рычаги, двери)allow / deny
use-itemsИспользование предметов (ведра, лодки, эндер-жемчуг)allow / deny
pvpPvP (игрок против игрока)allow / deny
mob-spawningСпавн мобов (всех типов)allow / deny

exitВыход из региона
