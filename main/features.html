⚙️ Возможности RegionMC

Полный обзор функционала плагина

🏰 Полная защита региона

  • Защита от разрушения/строительства
  • Защита от взрывов (криперы, TNT)
  • Контроль использования контейнеров, дверей
  • Предотвращение распространения огня и лавы

👥 Управление доступом

  • Владельцы (полный контроль)
  • Участники (могут строить/разрушать)
  • Поддержка оффлайн-игроков по UUID

🚩 36+ настраиваемых флагов

  • pvp, mob-spawning, entry, exit
  • use-items, chest-access, enderchest
  • time-lock (день/ночь/заморозка)
  • greeting, farewell, blockcmd, block-chat

🛠️ Инструменты работы

  • Палочка выделения (Wand)
  • Предмет проверки регионов (Stick)
  • Визуализация границ (частицы RGB)

📊 Гибкие лимиты

  • Максимальное количество регионов на игрока
  • Максимальный объём региона
  • Настройка по группам через limit.yml

💾 Производительность и хранение

  • SQLite (по умолчанию) или MySQL
  • Кэширование регионов
  • Автоматическое резервное копирование

🌍 Поддержка миров

  • Работа во всех мирах
  • Автоматическое создание глобального региона
  • Возможность отключить регионы в конкретном мире

🎨 Визуальные эффекты

  • Границы регионов (цветные частицы)
  • Уведомления о входе/выходе (ActionBar)
  • Цветные сообщения (HEX и градиенты)

🔌 PlaceholderAPI поддержка

  • 30+ плейсхолдеров
  • Информация о регионе, владельцах, участниках
  • Проверка прав и лимитов

🔧 API и расширяемость

  • События региона
  • Гибкая система флагов
  • Поддержка кликабельных сообщений (JSON)

⏰ Блокировка времени

  • Фиксация времени в регионе
  • Режимы: день, ночь, заморозка
  • Визуальный эффект для игроков

🔄 Автообновление конфигов

  • Автоматическое создание недостающих файлов
  • Сохранение пользовательских настроек
  • Поддержка нескольких языков (ru/en/ua)

🖥️ Поддержка консоли

  • Все команды доступны из консоли
  • Удобное администрирование без захода в игру

👑 Администрирование через admin-user

  • Назначение администраторов в Settings/main.yml
  • Полный доступ к командам без выдачи отдельных прав
