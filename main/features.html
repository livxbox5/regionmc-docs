⚙️ Возможности RegionMC

Полный обзор функционала плагина

🏰 Полная защита региона

  • Защита от разрушения/строительства
  • Защита от взрывов (криперы, TNT, ядра эндер-дракона)
  • Контроль использования контейнеров, дверей, рычагов, кнопок
  • Предотвращение распространения огня и лавы
  • Защита от эндерменов (поднятие блоков)

👥 Управление доступом

  • Владельцы (полный контроль, могут менять флаги)
  • Участники (могут строить/разрушать, если флаги allow)
  • Поддержка оффлайн-игроков по UUID
  • Возможность добавлять/удалять через команды

🚩 30+ настраиваемых флагов

  • pvp, mob-spawning, entry, exit
  • use-items, enderchest, chest-access
  • time-lock (день/ночь/заморозка)
  • greeting, farewell (сообщения)
  • blockcmd (запрет команд)

🛠️ Инструменты работы

  • Палочка выделения (любой материал из конфига)
  • Предмет проверки регионов (инфо-палочка)
  • Автоматическое определение предметов по материалу
  • Визуализация границ (частицы)

📊 Гибкие лимиты

  • Максимальное количество регионов на игрока
  • Максимальный объём региона (блоков³)
  • Настройка размеров (мин/макс)
  • Поддержка групп через права

💾 Производительность и хранение

  • SQLite (по умолчанию) или MySQL
  • Кэширование регионов, асинхронная загрузка
  • Автоматическое резервное копирование
  • Настройка интервала проверки региона

🌍 Поддержка миров

  • Работа во всех мирах (Overworld, Nether, End, кастомные)
  • Автоматическое создание глобального региона в каждом мире
  • Возможность отключить регионы в конкретном мире

🎨 Визуальные эффекты

  • Границы регионов (частицы, настраиваемый цвет и тип)
  • Уведомления о входе/выходе (action bar или чат)
  • Цветные сообщения (поддержка HEX и градиентов)

🔧 API и расширяемость

  • События региона (создание, удаление, изменение флагов)
  • PlaceholderAPI поддержка
  • REST API (опционально)
  • Поддержка других плагинов через API
