Основные настройки плагина (файл config.yml)

📁 Основные настройки

regions: max-regions-per-player: 5 allow-overlap: false default-priority: 0 size: min-size: 5 max-size: 100000 max-height: 256 naming: min-length: 3 max-length: 16 blocked-names: ["admin","global","world"]

🚩 Флаги по умолчанию

default-flags: build: "deny" break: "deny" interact: "allow" pvp: "allow" mob-spawning: "allow"

🛠️ Палочка выделения

selection-tools: selection-wand: enabled: true material: "WOODEN_AXE" name: "&6Топор для выделения" lore: ["&7ЛКМ - pos1", "&7ПКМ - pos2"]
