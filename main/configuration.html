⚙️ Конфигурация

Основные настройки плагина (файлы config.yml, main.yml, limit.yml)

⭐ ГЛАВНОЕ ПРАВИЛО: main.yml vs limit.yml

📌 Если limit.yml → Enable: true (лимиты ВКЛЮЧЕНЫ)
Тогда limit.yml имеет ПРИОРИТЕТ над main.yml для лимитов! 
# limit.yml
Enable: true                    # ← ключевой переключатель
default-max-regions: 10         # ← ЭТО значение будет использоваться

# main.yml  
max-regions-per-player: 5       # ← ИГНОРИРУЕТСЯ (так как Enable: true)
📌 Если limit.yml → Enable: false (лимиты ВЫКЛЮЧЕНЫ)
Тогда main.yml вступает в силу как запасной вариант.

⚠️ Важное замечание о флагах по умолчанию

Флаги по умолчанию, указанные в main.yml → regions.default-flags, НЕ ИСПОЛЬЗУЮТСЯ.
Вместо них применяются жёстко заданные значения в коде (RegionManager.setDefaultRegionFlags()). Для обычных регионов это:
build=deny, break=deny, place=deny, interact=allow, use-items=deny, pvp=allow, mob-spawning=allow, explosions=deny, creeper-explosion=deny, tnt-explosion=deny, time-lock=allow, entry=allow, exit=allow, chest-access=deny, enderchest=deny, block-chat=allow, enter-notify=deny, exit-notify=deny.

📊 Лимиты регионов (limit.yml)

⚠️ ВАЖНО: Этот файл имеет ПРИОРИТЕТ над main.yml для лимитов!
Если Enable: true - используются настройки из limit.yml.
Если Enable: false - используются настройки из main.yml. 
# ГЛОБАЛЬНОЕ ВКЛЮЧЕНИЕ/ВЫКЛЮЧЕНИЕ
Enable: true

# НАСТРОЙКИ РАЗМЕРОВ РЕГИОНОВ
size-limits:
  enable-size-checks: true
  min-size: 5
  max-size: 100000
  max-height: 256

# НАСТРОЙКИ КОЛИЧЕСТВА РЕГИОНОВ
default-max-regions: 5

# ПРИОРИТЕТ ПРОВЕРКИ
check-priority: permission-first

# ЛИМИТЫ ПО ПЕРМИШЕНАМ
permission-limits:
  - permission: "regionmc.limit.10000"
    limit: 10000
  - permission: "regionmc.limit.20000"
    limit: 20000
  - permission: "regionmc.limit.50000"
    limit: 50000
  - permission: "regionmc.limit.100000"
    limit: 100000
  - permission: "regionmc.limit.unlimited"
    limit: -1
  
  - permission: "regionmc.maxregions.5"
    max-regions: 5
  - permission: "regionmc.maxregions.10"
    max-regions: 10
  - permission: "regionmc.maxregions.20"
    max-regions: 20
  - permission: "regionmc.maxregions.unlimited"
    max-regions: -1

# ЛИМИТЫ ПО ГРУППАМ
group-limits:
  default:
    size: 26000
    max-regions: 5
  vip:
    size: 32000
    max-regions: 10
  premium:
    size: 40000
    max-regions: 20
  elite:
    size: 50000
    max-regions: 30
  admin:
    size: -1
    max-regions: -1

default-limit: 100000
💡 limit: -1 = безлимитно | max-height: -1 = безлимитная высота
