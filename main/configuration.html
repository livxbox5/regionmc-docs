⚙️ Конфигурация

Основные настройки плагина (файлы config.yml, main.yml, limit.yml)

📁 Основные настройки (config.yml)

settings:
  default-language: "ru"        # Язык плагина (ru/en)
  debug: false                   # Режим отладки
  auto-update-config: true       # Автообновление конфига
  save-regions-individually: true
  auto-save-interval: 6000       # Интервал автосохранения (тики)
  max-regions-per-player: 5      # Лимит регионов по умолчанию
  enable-global-regions: true    # Глобальные регионы
  creeper-cancel-effects: true
  auto-time-sync: true
  notify-owners-on-entry: true
  notify-owners-on-exit: false

updates:
  check-for-updates: true
  resource: "modrinth"
  check-interval: 24             # Часы
  notify-admins: true

⭐ ГЛАВНОЕ ПРАВИЛО: main.yml vs limit.yml

📌 Если limit.yml → Enable: true (лимиты ВКЛЮЧЕНЫ)
Тогда limit.yml имеет ПРИОРИТЕТ над main.yml для лимитов! 
# limit.yml
Enable: true                    # ← ключевой переключатель
default-max-regions: 10         # ← ЭТО значение будет использоваться

# main.yml  
max-regions-per-player: 5       # ← ИГНОРИРУЕТСЯ (так как Enable: true)
📌 Если limit.yml → Enable: false (лимиты ВЫКЛЮЧЕНЫ)
Тогда main.yml вступает в силу как запасной вариант! 
# limit.yml
Enable: false                   # ← лимиты отключены

# main.yml
max-regions-per-player: 5       # ← ЭТО значение используется

📁 Полная конфигурация (main.yml)


version: "1.0.0"

  default-priority: 0            # Приоритет новых регионов

  naming:
    blocked-names:
      - "admin"
      - "global"
      - "world"
      - "server"

      - "mine"
      - "protected"

    max-length: 16

    allow-spaces: false

  # =============================================
  regions-warn:
    # Включить/выключить систему предупреждений
    enabled: true

    entry-warning: true
    entry-warning-message: "&c⚠ Вы вошли в регион &e{region}&c! У вас нет прав!"

    break-warning: true
    break-warning-message: "&c⚠ Вы не можете ломать блоки в регионе &e{region}&c!"

    chest-warning: true
    chest-warning-message: "&c⚠ Вы не можете открывать контейнеры в регионе &e{region}&c!"
    
    # Предупреждение при попытке использовать предметы
    use-warning: true
    use-warning-message: "&c⚠ Вы не можете использовать предметы в регионе &e{region}&c!"

    pvp-warning: true
    pvp-warning-message: "&c⚠ PvP отключен в регионе &e{region}&c!"
    
    # Задержка между предупреждениями (в секундах)
    warn-cooldown: 5

  # =============================================
  # ФЛАГИ ПО УМОЛЧАНИЮ
  # =============================================
  default-flags:

    break: "deny"
    place: "deny"
    interact: "allow"
    use-items: "allow"
    pvp: "allow"
    mob-spawning: "allow"
    explosions: "deny"
    creeper-explosion: "deny"
    tnt-explosion: "deny"
    time-lock: "allow"
    entry: "allow"
    exit: "allow"
    chest-access: "allow"
    enderchest: "allow"

  # =============================================
  # НАСТРОЙКИ ИНСТРУМЕНТОВ
  # =============================================

📊 Лимиты регионов (limit.yml)

⚠️ ВАЖНО: Этот файл имеет ПРИОРИТЕТ над main.yml для лимитов!
Если Enable: true - используются настройки из limit.yml.
Если Enable: false - используются настройки из main.yml. 
# =============================================
# ГЛОБАЛЬНОЕ ВКЛЮЧЕНИЕ/ВЫКЛЮЧЕНИЕ
# =============================================
# true  - все лимиты ВКЛЮЧЕНЫ (limit.yml имеет приоритет)
# false - все лимиты ОТКЛЮЧЕНЫ (используется main.yml)
Enable: true

# =============================================
# НАСТРОЙКИ РАЗМЕРОВ РЕГИОНОВ
# =============================================
size-limits:
  enable-size-checks: true       # Включить проверку размеров
  min-size: 5                    # Минимальный размер (блоков)
  max-size: 100000               # Максимальный размер (блоков)
  max-height: 256                # Максимальная высота

# =============================================
# НАСТРОЙКИ КОЛИЧЕСТВА РЕГИОНОВ
# =============================================
default-max-regions: 5           # Лимит по умолчанию

# =============================================
# ПРИОРИТЕТ ПРОВЕРКИ
# =============================================
check-priority: permission-first  # permission-first / group-first

# =============================================
# ЛИМИТЫ ПО ПЕРМИШЕНАМ
# =============================================
permission-limits:
  - permission: "regionmc.limit.10000"
    limit: 10000
  - permission: "regionmc.limit.20000"
    limit: 20000
  - permission: "regionmc.limit.50000"
    limit: 50000
  - permission: "regionmc.limit.100000"
    limit: 100000
  - permission: "regionmc.limit.unlimited"
    limit: -1
  
  - permission: "regionmc.maxregions.5"
    max-regions: 5
  - permission: "regionmc.maxregions.10"
    max-regions: 10
  - permission: "regionmc.maxregions.20"
    max-regions: 20
  - permission: "regionmc.maxregions.unlimited"
    max-regions: -1

# =============================================
# ЛИМИТЫ ПО ГРУППАМ
# =============================================
group-limits:
  default:
    size: 26000
    max-regions: 5
  vip:
    size: 32000
    max-regions: 10
  premium:
    size: 40000
    max-regions: 20
  elite:
    size: 50000
    max-regions: 30
  admin:
    size: -1
    max-regions: -1

# =============================================
# ЛИМИТ РАЗМЕРА ПО УМОЛЧАНИЮ
# =============================================
default-limit: 100000

# =============================================
# СПЕЦИАЛЬНЫЕ ЗНАЧЕНИЯ
# =============================================
# limit: -1          → Безлимитно
# limit: 0           → Безлимитно
# max-height: -1     → Безлимитная высота
# min-size: -1       → Безлимитный минимум
💡 limit: -1 = безлимитно | max-height: -1 = безлимитная высота

📝 Примеры настройки лимитов

# Пример 1: Обычный игрок (без прав)
# - Размер: до 100 000 блоков
# - Количество: до 5 регионов
# - Высота: до 256 блоков

# Пример 2: VIP игрок (с правами)
# Права: regionmc.maxregions.10 и regionmc.limit.50000
# - Размер: до 50 000 блоков
# - Количество: до 10 регионов
# - Высота: до 256 блоков

# Пример 3: Безлимитная высота
max-height: -1

# Пример 4: Безлимитный размер
max-size: -1

# Пример 5: Безлимитное количество
default-max-regions: -1
