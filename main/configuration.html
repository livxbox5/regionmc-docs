⚙️ Конфигурация

Основные настройки плагина (файл config.yml)

📁 Основные настройки (settings)

settings:
  default-language: "ru"
  debug: false
  auto-update-config: true
  save-regions-individually: true
  auto-save-interval: 6000
  max-regions-per-player: 5
  enable-global-regions: true
  creeper-cancel-effects: true
  auto-time-sync: true
  notify-owners-on-entry: true
  notify-owners-on-exit: false

🔄 Обновления (updates)

updates:
  check-for-updates: true
  resource: "modrinth"
  check-interval: 24
  notify-admins: true

📐 Ограничения регионов

regions:
  size:
    min-size: 5
    max-size: 100000
    max-height: 256
  naming:
    min-length: 3
    max-length: 16
    blocked-names: ["admin","global","world"]

🚩 Флаги по умолчанию (default-flags)

default-flags:
  build: "deny"
  break: "deny"
  interact: "allow"
  pvp: "allow"
  mob-spawning: "allow"
💡 Эти флаги применяются ко всем новым регионам при их создании.

🛠️ Инструменты выделения (selection-tools)

selection-tools:
  selection-wand:

    durability: -1

💾 База данных (database)

database:
  type: "sqlite"   # или "mysql"
  mysql:
    host: "localhost"
    database: "regionmc"
    username: "root"
    password: "password"
💡 SQLite подходит для небольших серверов. MySQL — для крупных проектов.
