📋 История изменений

Список последних обновлений плагина RegionMC

v2.1 (2026-06-21)

✅ Добавлено

  • Полная поддержка консоли — все команды доступны из консоли сервера
  • Система admin-user в Settings/main.yml — администраторы получают все права
  • Новый класс PermissionUtil для централизованной проверки прав
  • Флаги enter-notify и exit-notify для уведомлений о входе/выходе
  • Категория ADMIN в DebugUtils для отладки
  • Метод getDefaultLimit() в RegionManager стал публичным для консоли

🛠️ Исправлено

  • Удалён дублирующийся метод getDefaultLimit() в RegionManager
  • BlockChatFlag теперь использует современный regionManager.isAdminBypass() вместо устаревшего API
  • Исправлены вызовы LanguageManager.getMessage() в RegionNotifierCommand
  • Устранены все предупреждения компиляции (deprecated API, unchecked операции)
  • Во всех подкомандах заменены прямые player.hasPermission() на PermissionUtil.hasPermission()

📝 Документация

  • Обновлены страницы документации: команды, права, флаги, конфигурация, PlaceholderAPI
  • Добавлена информация о приоритете limit.yml над main.yml
  • Уточнены значения флагов по умолчанию (из кода)
