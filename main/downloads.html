📋 Системные требования

🎮 Сервер

  • Paper / Purpur / Spigot 1.16+
  • Java 17 или выше
  • PlaceholderAPI (опционально)
  • Vault (опционально, для экономики)

💾 Ресурсы

  • RAM: от 50 МБ на 100 регионов
  • Диск: от 5 МБ + данные регионов
  • SQLite или MySQL

🛠️ Установка

1. Скачайте последнюю версию .jar файла 2. Поместите файл в папку plugins/ вашего сервера 3. Перезапустите сервер 4. Настройте конфиг в plugins/RegionMC/config.yml 5. Готово! Используйте /rg help для начала работы
💡 После установки выдайте права игрокам через LuckPerms или другой менеджер прав.
Пример: /lp user Steve permission set regionmc.command.claim true
