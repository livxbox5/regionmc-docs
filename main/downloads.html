🛠️ Установка

1. Скачайте последнюю версию .jar файла

2. Поместите файл в папку plugins/ вашего сервера

3. Перезапустите сервер

4. Настройте конфиги в папке plugins/RegionMC/

5. Готово! Используйте /rg help для начала работы

💡 После установки выдайте права игрокам через LuckPerms.
Пример: /lp user Steve permission set regionmc.command.claim true
